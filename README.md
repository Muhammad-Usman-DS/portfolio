# Personal Portfolio Website

A modern, responsive portfolio website built to showcase projects, technical skills, and professional experience in a clean and structured format. 

🔗 **Live Website:** https://www.syedusmann.com

---

## Overview

This portfolio serves as a central hub for presenting:
- End-to-end projects in data science, analytics, and web development
- Technical tools and technologies used across projects
- Live demos and source code links
- A clear and professional personal brand

The website follows modern frontend and deployment standards with Continuous Integration and Continuous Deployment.

---

## Key Features

- Responsive UI optimized for desktop and mobile
- Interactive project cards with GitHub and live demo links
- Clean component based architecture
- Deployed with CI/CD for automatic updates on code changes
- Custom domain with HTTPS and SSL enabled

---

## Tech Stack

### Frontend
- React
- Vite
- JavaScript (ES6+)
- HTML5 / CSS3
- Tailwind CSS
- Three.js (3D & visual elements)
- Framer Motion (animations)

### Tooling & Deployment
- Git & GitHub (version control)
- Vercel (hosting & CI/CD)
- Global CDN for fast load times

---

## Deployment Workflow

This project uses a modern production deployment pipeline:

1. Code is pushed to the `main` branch on GitHub
2. Vercel automatically triggers a build
3. The site is rebuilt and deployed globally
4. Changes go live without downtime
